# 常用经典字体

| 分类   | 字体名称 | 英文名                            | 特点 / 用途                  |
| ------ | -------- | --------------------------------- | ---------------------------- |
| 宋体系 | 宋体     | SimSun                            | 横细竖粗，传统正文排版       |
|        | 新宋体   | NSimSun                           | 屏幕显示优化版宋体           |
|        | 华文中宋 | STZhongsong                       | 现代化宋体风格，报告/正文    |
| 黑体系 | 黑体     | SimHei                            | 标题、标识                   |
|        | 微软雅黑 | Microsoft YaHei                   | UI / 屏幕阅读                |
|        | 方正黑体 | FZHei                             | 广告、海报                   |
|        | 华文细黑 | STXihei                           | 细黑体，正文/UI              |
| 楷体系 | 楷体     | KaiTi                             | 注释、书法风格               |
|        | 标楷体   | DFKai-SB                          | 打印友好                     |
| 仿宋系 | 仿宋     | FangSong                          | 公文、正式文件               |
|        | 华文仿宋 | STFangsong                        | 仿宋风格，现代系统常见       |
| 隶书系 | 隶书     | LiSu                              | 传统文化类标题               |
|        | 方正隶书 | FZLiShu                           | 装饰性标题                   |

|        | 华文彩云 | STCaiyun                          | 装饰、海报                   |
|        | 华文新魏 | STXinwei                          | 魏碑风格，书法感             |
| 思源系 | 思源宋体 | Source Han Serif / Noto Serif CJK | 现代开源宋体，多语言一致性强 |
|        | 思源黑体 | Source Han Sans / Noto Sans CJK   | 开源、跨平台、UI/正文通用    |

## **二、英文字体（Latin / English Fonts）**

| 分类              | 字体名称        | 特点 / 用途                |
| ----------------- | --------------- | -------------------------- |
| 衬线 Serif        | Times New Roman | 学术论文、正式文档         |
|                   | Georgia         | 屏幕友好衬线               |
|                   | Garamond        | 古典书籍排版               |
|                   | Noto Serif      | 与思源宋体配套             |
| 无衬线 Sans-serif | Arial           | 通用字体                   |
|                   | Helvetica       | 现代设计标准               |
|                   | Calibri         | Office 默认                |
|                   | Noto Sans       | 与思源系列配套，跨语言一致 |
| 等宽 Monospace    | Courier New     | 代码、表格                 |
|                   | Consolas        | 终端、编程                 |
| 手写 Script       | Brush Script MT | 装饰性标题                 |
| 装饰 Display      | Impact          | 海报、广告                 |
